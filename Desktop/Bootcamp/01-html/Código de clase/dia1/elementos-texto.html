Elementos de texto

La etiqueta p define un párrafo.

Es el elemento más importante.

Cada párrafo debe ir en su propio elemento p.

La etiqueta em denota énfasis

La etiqueta strong denota importancia

La etiquea br se utiliza unicamente para saltos de linea forzados (por ejemplo en poemas)

Yo quiero ser llorando el hortelano
de la tierra que ocupas y estercolas,
compañero del alma, tan temprano.

La etiqueta span no tiene valor semántico. Sirve para seleccionar una parte con CSS o JS

Blockquote define una cita

Words can be like X-rays, if you use them properly—they’ll go through anything. You read and you’re pierced.

El elemento cite es para indicar la fuente de la cita

—Aldous Huxley, Brave New World
